Beneficios

¿Porque deberisa contratarme?

Cuando construyo o clono proyectos web, me adhiero firmemente a estos principios que hacen que mis sitios web sean únicos y escalables.

SEO(Search Engine Optimization)

Empleo las mejores prácticas de SEO como "Semantic HTML" y más en sitios web y proyectos en los que trabajo para mejorar la visibilidad del sitio y la clasificación en Google.

Responsive Design

Me apasiona la capacidad responsive y hacer que los sitios web que construyo se vean bien en diferentes dispositivos, navegadores y plataformas cruzadas utilizando tanto el enfoque "mobile first" como las "media queries" de escritorio.

Código reutilizable y mínimo

Hago todo lo posible para escribir el código mínimo posible usando herramientas como Variables CSS y escribiendo clases de utilidad personalizadas.
